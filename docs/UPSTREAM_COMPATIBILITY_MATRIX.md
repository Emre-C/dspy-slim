# Upstream Compatibility Matrix

This is the first-pass audit matrix for keeping `dspy-slim` behaviorally faithful to upstream DSPy on its supported surface.

Use this file as the working checklist before changing user-visible behavior and before preparing upstream issues or PRs from local findings.

## Audit Status

- Upstream stable references refreshed in `../tmp/`: `stanfordnlp/dspy` `3.1.3` and `gepa-ai/gepa` `v0.1.1`, verified directly for the surfaces below
- Local reference: this checkout as of 2026-04-14
- Scope of this refresh pass: the kept `dspy-slim` boundaries (`Predict`, `ChainOfThought`, `ReAct`, `RLM`, `GEPA`) plus the supporting functions and repo docs that define those surfaces
- This matrix remains the stable-release compatibility artifact; post-`3.1.3` `main`-branch mergeability work is tracked separately in [`UPSTREAM_FORWARD_COMPATIBILITY.md`](UPSTREAM_FORWARD_COMPATIBILITY.md)
